## ðŸ“¹ RECORDING TIPS

### Setup
1. Use screen recording software (OBS, QuickTime, etc.)
2. Record at 1920x1080 minimum
3. Use high-quality microphone
4. Record in quiet environment

### Demo Recording
1. Open `frontend/demo.html` in browser
2. Make window fullscreen
3. Start screen recording
4. Follow shot list above
5. Move cursor smoothly, not too fast
6. Pause between clicks for clarity

### Voiceover Tips
1. Speak clearly and naturally
2. Use emotional inflection (especially emergency scene)
3. Pause for emphasis
4. Re-record if you mess up (edit later)
5. Sound confident but empathetic

### Editing
1. Sync voiceover with screen recording
2. Add music bed (low volume, don't overpower voice)
3. Add text overlays for key points:
   - "25% of seniors live alone"
   - "Emergency detected in <1 second"
   - "Alerts sent instantly"
4. Color grade for warm, professional look
5. Export at 1080p, 30fps
